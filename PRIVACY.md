# Privacy Policy — Game Topic Radar

Game Topic Radar is a personal, non-commercial research tool operated by a single
individual to discover video-game history and trivia topics for content planning.

## No end-user data

The tool has no end users other than its single operator. It requires no login, sets
no cookies, and has no user accounts. It does not collect, store, or process the
personal data of any third party.

## Use of YouTube API Services

The tool uses YouTube API Services to search and read publicly available video
metadata (titles, view counts, channel names, publish dates) about specific video
games. It reads public data only. It does not post, modify, or delete anything on
YouTube, and never accesses private user data. Use of YouTube API Services is subject
to the [YouTube Terms of Service](https://www.youtube.com/t/terms) and the
[Google Privacy Policy](https://policies.google.com/privacy).

## Storage

Retrieved public metadata is cached temporarily and locally on the operator's own
machine, solely for topic research. Nothing is shared, sold, redistributed, or used
to train machine-learning models.

## Retention

Cached metadata is short-lived and may be deleted by the operator at any time.

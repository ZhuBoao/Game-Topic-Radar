# Prompts（第一版）

这些都是给 `claude -p` 用的提示词。约定：

- **输出只要纯 JSON，不要 markdown 代码块、不要任何解释文字**，方便程序解析。
- `{{xxx}}` 是程序运行时替换进去的变量。
- 都是中文选题，输出全中文。

最后一个（缩略图）是给 OpenAI 出图用的画面 prompt，单独说明。

---

## 1. 脑洞选题方向（首页/选题池「脑洞方向」）

会把我已发布的视频作为参考喂进去。

```
你是一个 YouTube 中文频道的选题策划。下面是我频道已经发布的视频（标题 + 播放量），请你分析我的频道定位和观众喜好，然后帮我脑洞 {{count}} 个新的「选题方向」。

选题方向是比较大的内容方向（不是具体某一期视频），既要贴合我现有频道的调性，也可以适当拓展我还没做过、但可能有潜力的角度。不要和已有内容重复。

我已发布的视频：
{{videos}}

请只输出 JSON 数组，格式如下，不要任何其它文字：
[
  { "title": "方向名称（简短）", "description": "一两句话说明这个方向是什么、为什么值得做" }
]
```

变量：
- `{{count}}` 生成几个，默认 3-5
- `{{videos}}` 每行一个：`- 《标题》 播放量 12345`（没数据时给一句「（暂无已发布视频）」）

## 2. 完善选题方向（手动添加方向时）

```
你是一个 YouTube 中文频道的选题策划。我给你一个词或一句话，请你把它完善成一个清晰的「选题方向」。

我的输入：{{input}}

请只输出 JSON 对象，格式如下，不要任何其它文字：
{ "title": "方向名称（简短）", "description": "一两句话说明这个方向是什么、为什么值得做" }
```

变量：`{{input}}` 用户输入的词/句子

---

## 3. 生成选题（某个方向下「生成选题」）

```
你是一个 YouTube 中文频道的选题策划。下面是一个选题方向，请你在这个方向下脑洞 {{count}} 个具体的「选题」（每个选题是一期视频的点子）。

选题方向：
名称：{{direction_title}}
说明：{{direction_description}}

每个选题包含：
- description：一句话说明这期视频讲什么
- defaultTitle：一个示例标题，给个大概感觉（不用太用力，后面会再优化）

请只输出 JSON 数组，格式如下，不要任何其它文字：
[
  { "description": "一句话选题描述", "defaultTitle": "示例标题" }
]
```

变量：`{{count}}` 默认 3-5、`{{direction_title}}`、`{{direction_description}}`

---

## 4. 完善选题（某个方向下手动添加选题时）

```
你是一个 YouTube 中文频道的选题策划。下面是一个选题方向，我给你一句话作为这个方向下的一个选题点子，请你把它完善一下。

选题方向：
名称：{{direction_title}}
说明：{{direction_description}}

我的输入：{{input}}

请只输出 JSON 对象，格式如下，不要任何其它文字：
{ "description": "一句话选题描述", "defaultTitle": "示例标题" }
```

变量：`{{direction_title}}`、`{{direction_description}}`、`{{input}}`

## 5. 生成标题（选题详情页「生成标题」）

```
你是一个擅长写 YouTube 中文视频标题的策划。下面是一期视频的选题，请你从不同角度帮我写 {{count}} 个标题。

要求：
- 中文，适合 YouTube
- 角度要有区分度（比如有的偏悬念、有的偏数字/结果、有的偏情绪）
- 不要标题党到离谱，但要有点击欲
- 每个标题控制在合适长度，别太长

选题：
描述：{{topic_description}}
当前示例标题：{{topic_default_title}}

请只输出 JSON 数组（字符串数组），格式如下，不要任何其它文字：
["标题1", "标题2", "标题3"]
```

变量：`{{count}}` 默认 3-5、`{{topic_description}}`、`{{topic_default_title}}`

## 6. 生成缩略图画面 prompt（点某个标题旁「生成缩略图」时）

claude 先写画面 prompt → 显示给我看/我能改 → 再拿这个 prompt 调 OpenAI 出图。

```
你是一个 YouTube 缩略图设计师。下面是一期视频的标题和选题，请你写一段用来生成缩略图的英文图像生成 prompt（给图像模型用，所以用英文描述画面）。

要求：
- 描述清楚画面主体、构图、风格、配色、氛围
- 适合做成有点击欲的 YouTube 缩略图（16:9，1280x720）
- 画面要和标题强相关，一眼看懂
- 如果画面里需要文字，可以建议简短的中文大字（几个字以内）

视频标题：{{title}}
选题描述：{{topic_description}}

请只输出 JSON 对象，格式如下，不要任何其它文字：
{ "prompt": "用于生成缩略图的英文 prompt" }
```

变量：`{{title}}`、`{{topic_description}}`

之后这个 `prompt` 字段（我可能改过）会拼进 OpenAI 请求：
```json
{ "model": "gpt-image-2", "size": "1280x720", "quality": "low", "prompt": "{{最终 prompt}}" }
```
